# Lettaga Homepage — Features 스포트라이트 카피 (EN / KO 검토용)

각 제품: **헤드라인 + 불릿 3개**. 영문이 실제 게재본, 국문은 이해·검토용 번역입니다.
수정할 부분을 이 파일에 바로 표시하거나 알려주세요.

---

## 1. T-Sync — Synchronisation

**EN**
- Headline: Your Register Book, always in sync with the registry
- Real-time sync with ACRA & SSM
- Changes auto-recorded to your Register Book
- Spots registry changes you didn't make

**KO**
- 헤드라인: 등록기관과 항상 동기화되는 Register Book
- ACRA·SSM와 실시간 동기화
- 변경 사항을 Register Book에 자동 기록
- 내가 하지 않은 등록기관 변경도 감지

---

## 2. T-Link — Client Link

**EN**
- Headline: Clients update their own details — through one secure link
- One public deep link — no client login
- Incorporation & CDD submitted by the client
- Flows straight into drafting, T-Sign & e-Filing — end to end

**KO**
- 헤드라인: 보안 링크 하나로 고객이 직접 정보를 입력
- 공개 딥링크 하나 — 고객 로그인 불필요
- 설립 정보·CDD를 고객이 직접 제출
- 이어서 문서작성·T-Sign·e-Filing까지 — 엔드투엔드 연결

---

## 3. T-Doc AI — Document Generation

**EN**
- Headline: Use your own templates — AI fills them for every entity
- Upload your own templates — drag & drop DOCX or ZIP
- AI extracts every [[variable]] automatically
- Generate the same set for many entities at once

**KO**
- 헤드라인: 보유한 템플릿을 그대로 — AI가 엔터티별로 채움
- 보유 템플릿 직접 업로드 — DOCX·ZIP 드래그&드롭
- AI가 모든 [[변수]]를 자동 추출
- 여러 엔터티에 같은 세트를 한 번에 생성

---

## 4. T-Sign — e-Signature

**EN**
- Headline: Sign with clients 24/7 — built right in
- Built-in e-signature — nothing extra to buy
- Track every signer: viewed → signed
- Signed files auto-saved to T-Drive

**KO**
- 헤드라인: 고객과 24/7 전자서명 — 빌트인으로
- 내장 전자서명 — 추가 구매 없이
- 서명자별 상태 추적: 열람 → 서명 완료
- 서명 완료 파일은 T-Drive에 자동 저장

---

## 5. T-Drive — Cloud Drive

**EN**
- Headline: Every document filed to the right entity — automatically
- Signed docs & filing certificates auto-saved
- A drive per entity: Documents · Privacy · Transactions
- Secure cloud storage on Microsoft Azure

**KO**
- 헤드라인: 모든 문서가 알맞은 엔터티에 자동 보관
- 서명 문서·제출 증서를 자동 저장
- 엔터티별 드라이브: Documents·Privacy·Transactions
- Microsoft Azure 기반 보안 클라우드 저장

---

## 6. T-Filing — e-Filing

**EN**
- Headline: From draft to filed — in one click
- Forms pre-filled from your task
- Filed straight to ACRA & SSM
- Certificate returned, stage marked done

**KO**
- 헤드라인: 초안에서 제출까지 — 원클릭
- 태스크 데이터로 서식 자동 사전 입력
- ACRA·SSM에 바로 제출
- 증서 수령 후 단계 자동 완료 표시

---

## 7. AI XBRL — 🇸🇬 Singapore only · Add-on

**EN**
- Headline: AI tags your financials to ACRA's XBRL — automatically
- AI maps every line to ACRA's XBRL taxonomy
- Simplified or Full XBRL, validated automatically
- Ready to file with ACRA — available on the Marketplace

**KO**
- 헤드라인: AI가 재무제표를 ACRA XBRL로 자동 태깅
- AI가 각 항목을 ACRA XBRL 택소노미에 매핑
- Simplified/Full XBRL 자동 생성·검증
- ACRA 제출 준비 완료 — 마켓플레이스에서 제공
- 라벨: 🇸🇬 싱가포르 전용 · 애드온(별도 과금)

---

## 8. AR — Annual Return · 🇲🇾 Malaysia only

**EN**
- Headline: Annual Returns, prepared from your synced register data
- Auto-collected from each entity's Register
- Validated in SSM's MBRS (XBRL) format
- Confirm once, then lodge to SSM

**KO**
- 헤드라인: 동기화된 레지스터 데이터로 연차보고(AR) 자동 작성
- 엔터티별 Register에서 자동 수집
- SSM MBRS(XBRL) 형식으로 검증
- 한 번 Confirm 후 SSM에 제출
- 라벨: 🇲🇾 말레이시아 전용
